# web_agenzia_viaggi
Progetto di gruppo di TPSIT e Informatica

Il sito in questione è nominato come Wanderwave Travel, dove all'interno gestisce le prenotazioni per i viaggi all'estero. 
È dotato di un database che rende il sito dinamico, consentendo il filtraggio delle informazioni e il salvataggio delle prenotazioni.
